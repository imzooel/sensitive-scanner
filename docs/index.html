Sensitive Scanner

Offensive Security Tool for Detecting Sensitive Data in JS Files

Overview

SensitiveScanner is a Go-based tool designed to identify sensitive data (e.g., API keys, tokens) in JavaScript files. Built with a focus on bug bounty hunting and web security, it leverages Go's concurrency for efficient scanning. This tool is perfect for ethical hackers, security researchers, and developers looking to secure their applications.

Features

  • Detects sensitive data like API keys, Stripe keys, and tokens using regex patterns.
  • Supports concurrent scanning with adjustable concurrency (default: 5).
  • Scans single URLs or lists of URLs from a file.
  • Outputs results to the console or a JSON file.
  • Beautifies minified JS files for better detection.
  • Verbose mode for detailed logging.

Installation

1. Ensure you have Go (1.21+) installed.

2. Clone the repository:

git clone https://github.com/imzooel/sensitive-scanner.git
cd sensitive-scanner
go build -o sensitive-scanner

Usage

Scan a single URL:

./sensitive-scanner -url https://example.com/script.js

Scan multiple URLs from a file (urls.txt):

./sensitive-scanner -url-file urls.txt -concurrency 10 -output results.json

Sample Output:

[ALERT] Found sensitive data in https://example.com/script.js
Match: pk_live_abc123xyz456
Severity: Medium
Pattern: Stripe Key
